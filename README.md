# Modern Calculator — Dark Green UI

This project is designed to look closer to a native mobile calculator app.

## Structure
```text
Calculator_Pro_DarkGreen/
├── index.html
├── style.css
├── script.js
├── manifest.json
├── sw.js
├── README.md
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## GitHub Pages
1. Create a repository.
2. Upload all files to the repository root.
3. Go to Settings → Pages.
4. Choose Deploy from branch → main → / (root).

## Features
- Dark/Light mode
- Mobile-first layout
- Offline support
- Green accent buttons
- PWA-ready
